# 🐉 Dragon Ball API

Projeto desenvolvido com HTML, CSS e JavaScript consumindo a API pública de Dragon Ball.

##  Funcionalidades

- Listagem de personagens
- Busca de personagens por nome
- Cards responsivos
- Consumo de API com Fetch API
- Interface simples e responsiva

##  Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript
- Dragon Ball API

## 🔗 API utilizada

https://dragonball-api.com/api/characters?limit=58
